API documentation

User controller

Register

Description
Registers a user
Endpoint
/user/register
HTTP Method
POST
Bearer token
Not required

Example request body

{
    email:teszt@elek.hu,
    password:Teszt12345,
    firstname:John,
    lastname:Doe,
    gender:MALE,
}

Expected response 200

{
    token:jwt-token-here,
}

Errors

This endpoint uses the following error codes:

  • 400 Bad RequestThe email or password is not valid.
  • 403 ForbiddenDefault response. Happens on any other error.
  • 409 ConflictThe email address is already belongs to another user.
  • 500 Internal Server ErrorAn unexpected error occurred on the server.
